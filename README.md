# Guruvai Intern Assessment

Two problems implemented in Python as part of a software engineering internship assessment.

## Files

- `lru_cache.py` — LRU Cache implementation using a HashMap and Doubly Linked List
- `event_scheduler.py` — Event Scheduler with overlap detection and minimum room calculation

## Problems

**Problem 1: LRU Cache**
Implements `get(key)` and `put(key, value)` operations in O(1) time using a HashMap combined with a Doubly Linked List.

**Problem 2: Event Scheduler**
Implements `can_attend_all(events)` to detect overlapping events and `min_rooms_required(events)` to calculate the minimum rooms needed using a greedy min-heap approach.

## Notes

The detailed explanations, trade-off analysis, future-proofing discussion, and concurrency considerations are included in the submitted document.
